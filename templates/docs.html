Quick Guide

Terminal

Type Linux/Termux commands in the input field. Press Ctrl+Enter or click [EXEC]. Use ↑/↓ keys for history navigation.

Extra Keys

Symbol bar above terminal provides quick access to Tab, |, ~, $, >. CTRL+C stops running processes.

File Manager

Click [FILES] in header. Browse folders, click files to edit. Use [SAVE] to save changes. Upload from Android gallery or native file manager.

Global Search

In File Manager, click SEARCH. Search text across all project files. Results open files directly to matched lines.

PWA Install

Install as standalone Android app. Automatic prompt appears, or use "Add to Home Screen" from browser menu. Runs without address bar like native app.
