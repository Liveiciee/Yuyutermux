⚡ quick guide

▶︎ terminal

Ketik perintah Linux / Termux di input bawah. Tekan Ctrl + Enter atau klik tombol [ EXEC ]. Gunakan ↑ / ↓ pada extra keys untuk navigasi history.

⌨️ extra keys

Bar simbol di atas terminal: akses cepat Tab | ~ $ > dll. CTRL+C untuk menghentikan proses yang berjalan.

📂 file manager

Klik [ FILES ] di header terminal. Jelajahi folder, klik file untuk edit langsung. Gunakan tombol [ SAVE ] untuk menyimpan. Upload dari galeri Android / file manager native.

🔍 global search

Di dalam File Manager, klik tombol SEARCH. Cari teks di seluruh file project. Hasil pencarian bisa langsung membuka file dan scroll ke baris yang cocok.

📲 PWA install

Yuyutermux dapat diinstall sebagai aplikasi Android mandiri. Muncul prompt otomatis, atau pilih “Add to Home Screen” dari menu browser. Berjalan tanpa address bar seperti app native.

🛠️ tech stack & architecture

backend

Python 3 + Flask · Waitress WSGI · subprocess isolation · shlex.quote anti-injection

frontend

Vanilla ES modules · zero build · Service Worker (PWA) · responsive & offline fallback

storage

LocalStorage (history, prefs) · flat file system · no database overhead

ui / ux

Cyber-minimalist OLED theme · WeeChat flex layout · highlight.js syntax

target platform

Termux on Android · Chrome / Kiwi · low resource & battery efficient

security

Path traversal guard · max file size 1MB · command sanitization · CORS-restricted
